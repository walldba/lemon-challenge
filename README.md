## Usage

To install all packages use the command:

```bash
yarn install
```

To run unit tests and check coverage:

```bash
yarn test:cov
```

To run e2e tests:

```bash
yarn test:e2e
```

To run the app use:

```bash
yarn start
```

This app run on port 3000.

To check the app health:

```bash
curl --location --request GET 'http://localhost:3000/'
```
